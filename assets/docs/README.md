# Documentos da Ouvidoria da Mulher

Este diretório contém os materiais informativos e documentos para download do site.

## Arquivos Esperados

Por favor, faça upload dos seguintes PDFs para este diretório:

1. **plano_trabalho_extensao.pdf** - Plano de Trabalho do Projeto de Extensão
2. **butique_solidaria.pdf** - Informações completas sobre a Butique Solidária
3. **ouvidoria_mulher.pdf** - Cartilha geral da Ouvidoria da Mulher
4. **ouvidoria_mulher_demandas.pdf** - Demandas e conteúdos para comunicação

## Como Subir os Arquivos

1. Acesse o repositório no GitHub
2. Navegue até a pasta `assets/docs/`
3. Clique em "Add file" → "Upload files"
4. Arraste os PDFs ou selecione-os do seu computador
5. Adicione uma mensagem de commit (ex: "Adicionar documentos informativos")
6. Clique em "Commit changes"

## Nota

Enquanto os PDFs não forem enviados, os links de download no site apontarão para este README. Após o upload, atualize os links no `index.html` se necessário.
